# 環境設定

左メニューにございます `環境設定` 項目の機能についてご紹介します。

## ユーザー設定

Ripple アカウントのパスワード変更ができます。

言語設定を英語、日本語から選択いただけます。

## 支払いアカウント設定

AWSの支払いアカウント（Payer Account）の追加、削除ができます。

## 請求書設定

全ての請求グループで共通な設定を行います。
